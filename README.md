---

<h2>📌 Project Overview</h2>

<p>
This project presents an interactive <b>Road Accident Analysis Dashboard</b> created using Microsoft Excel.  
The dashboard analyzes accident data to identify trends, casualty distribution, and vehicle involvement.
</p>

<p>
The goal of this project is to demonstrate <b>data cleaning, analysis, and dashboard creation skills</b> using Excel.
</p>

---

<h2>📊 Dataset Information</h2>

<ul>
<li>Accident Date</li>
<li>Vehicle Type</li>
<li>Casualty Severity</li>
<li>Road Type</li>
<li>Location</li>
<li>Number of Casualties</li>
</ul>

<p>
The dataset was cleaned and analyzed using Excel before building the dashboard.
</p>

---

<h2>📈 Dashboard Features</h2>

<ul>
<li>Total Accidents KPI</li>
<li>Total Casualties KPI</li>
<li>Fatal Casualties</li>
<li>Serious Casualties</li>
<li>Monthly Accident Trend</li>
<li>Vehicle Type Analysis</li>
<li>Casualty Severity Distribution</li>
<li>Road Type Analysis</li>
</ul>

<h2>🛠 Tools Used</h2>

<li>Data Visualization</li>
</ul>

---

<h2>🔍 Key Insights</h2>

<ul>
<li>Some vehicle types are involved in accidents more frequently.</li>
<li>Accident cases vary across different months.</li>
<li>Most casualties fall under the slight injury category.</li>
<li>Road type influences accident frequency.</li>
</ul>
